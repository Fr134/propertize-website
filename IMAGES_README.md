## ⚙️ Specifiche tecniche consigliate

- **Formato**: JPG per foto, PNG per logo (con trasparenza)
- **Hero slideshow**: almeno 1920×1080px, qualità alta
- **Card/Discover**: almeno 800×600px
- **Gift card**: almeno 600×400px
- **Peso**: max 200-300 KB per foto (comprimere con TinyPNG o Squoosh)
- **Naming**: tutto minuscolo, trattini al posto degli spazi

---

## 🔗 Come usare nell'HTML

```html
<!-- Esempio URL raw da GitHub -->
<img src="https://raw.githubusercontent.com/Fr134/propertize-website/main/images/hero-golfo-aranci.jpg" alt="Golfo Aranci, Sardegna">
```
